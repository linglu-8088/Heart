# Heart Python Versions

Recommended main version: `li_xun_heart.py`

Run it with:

```powershell
python li_xun_heart.py
```

Controls:

- `Esc`: close the window
- `Space` / `R`: regenerate the particle layout
- Left click: regenerate the particle layout

Notes:

- `li_xun_heart.py` is the maintained entry point.
- Other `*_optimized.py` and `*_refactored.py` files are retained as legacy experiments for comparison.
